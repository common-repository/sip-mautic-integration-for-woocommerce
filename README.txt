Sync customers from WooCommerce to Mautic CRM.

== Description ==

This plugin allows you to sync existing and new WooCommerce customers to Mautic, optionally adding tags.

<h3>Key Features</h3>

The plugin is packaged with the following features:

* **Sync new customer and past customer information to Mautic**.
* **Optionally specify tags to add to these contacts, including product SKU and product name**.
